# Ανίχνευση Ανωμαλιών με ECOD

Αυτό το notebook παρουσιάζει την εφαρμογή του αλγορίθμου ECOD, για την ανίχνευση ανωμαλιών σε δεδομένα.

Το αρχικό notebook χρησιμοποιούσε τον αλγόριθμο IForest από τη βιβλιοθήκη PyOD. Στο παρόν αρχείο, ο IForest έχει αντικατασταθεί με τον αλγόριθμο ECOD.

Για την εκτέλεση του notebook εγινε εγκατάσταση της βιβλιοθήκης: pip install pyod.
